# Firmware Static Analysis Tools

Read-only scripts and helper sources used to inventory firmware updater packages and inspect managed/native updater code offline.

Excluded from GitHub-facing content:

- extracted firmware/update packages
- vendor DLL/EXE files
- Mono.Cecil binary dependencies
- raw logs and generated disassembly dumps

The public summary is in `docs/research-log/legacy-research-summary.md`; curated technical findings are in `docs/analysis/`.
