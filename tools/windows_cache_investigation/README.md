# Windows Cache Investigation Tools

Scripts and compact context files from the investigation into whether GIGABYTE System32 services or Windows-side caches were re-applying stale LCD media.

Excluded from GitHub-facing content:

- huge CSV search outputs
- registry dumps
- WER/DMP files
- generated contact-sheet images
- raw logs

The public summary is kept in `docs/research-log/legacy-research-summary.md`.
